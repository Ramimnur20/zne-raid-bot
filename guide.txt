ZNE Start Guide
Welcome to the ZNE Start Guide, this file will teach you
everything you need to know to get the zne raid bot up and working!

STEP 1 - setup venv

command:
python3 -m venv venv
source venv/bin/activate

STEP 2 - install dependencies

command:
pip install -r requirements.txt


STEP 3 - edit config

command:
config.toml

STEP 4 - start bot
python main.py
